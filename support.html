Support

Practical checks when something is not working.

Start with the relevant section below. Most typing, clicking, formatting, and playback features work without internet or an API key.

Troubleshooting topics

Use these checks before reinstalling or changing saved settings.

The app does not open

  • Install the current release from Microsoft Store.
  • Restart Windows after a pending system update.
  • Open Microsoft Store Library and check for app updates.
  • Confirm Windows 10 version 1809 or later is installed.

AI Writing is unavailable

  • Confirm AI Writing Mode is enabled.
  • Paste an active OpenAI API key into the masked field.
  • Check the internet connection.
  • Review API account status, rate limits, and usage costs.

Formatting does not appear

  • Rich HTML Paste depends on the receiving app.
  • Try Copy Rich Format before pasting.
  • Use Plain Text Typing when the target does not support HTML clipboard content.

Automation is running

  • Use Pause to keep the current position.
  • Use Resume to continue the same job.
  • Use Stop to cancel the active workflow.
  • Only one typing, clicking, or playback job runs at a time.

Common questions

Quick answers

Do I need an API key to use Autotyper Pro?

No. Auto Typer, Auto Clicker, Rich Text, Task Recorder, Human-like Typing, Mistake Mode, themes, and settings work without an API key. A user-provided key is required only for optional AI Writing.

Why did rich formatting become plain text?

The receiving app chose the plain-text clipboard format or does not support HTML paste. Plain Text Typing works almost everywhere; formatted output depends on the target app.

What is the difference between Pause and Stop?

Pause keeps the current job position so it can resume. Stop cancels the job; the next Start begins a new workflow.
