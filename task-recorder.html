Task Recorder

Record once. Replay with control.

Capture a short sequence of supported mouse and keyboard actions, review the timeline, and choose how playback should run.

Playback speedRepeat countPause and resumeVisible stop control

How it works

A short, visible workflow.

Recording never starts automatically. You choose when to begin, when to stop, and when to play the sequence.

1

Start recording

Select Record, then perform the supported actions you want to capture.

2

Review the timeline

Stop recording and inspect the ordered list of actions and delays.

3

Set playback

Choose a speed multiplier, repeat count, or a clearly indicated continuous loop.

4

Stay in control

Play, pause, resume, or stop while progress remains visible.

Playback options

Timing that matches the task.

Speed changes the recorded delays; repeats apply to the complete sequence.

0.25× to 2×

Slow down or speed up the recorded timing.

Repeat count

Run the sequence a selected number of times.

Continuous loop

Repeat until you use the visible Stop control.

Pause and resume

Hold the current playback position and continue later.

Privacy and safety

Designed for user-controlled workflows.

Use Task Recorder for actions you are authorised to automate. Avoid recording passwords, API keys, or other sensitive input.

  • Recording requires an explicit start action
  • Playback requires an explicit play action
  • Only one automation workflow runs at a time
  • Closing the app stops playback and releases active input
  • Saved recordings use local app storage and are not uploaded by the recorder
i

Keep Stop available.
Continuous playback remains visibly active until you stop it. Clear a recording when you no longer need it.
